Set Up Hugo in Git Hub Pages

October 22, 2023

Create a repository to hold the source code of your blogs#

Create a repository of Git Hub Pages#

Create a new Hugo project in your local machine#

cd ~/Projects
hugo new site <site name>
cd <site name>
git init
git remote add origin <repository URL of your blogs>
git add .
git commit -m "Initiate a new hugo project"
git push origin main

Choose theme for your blog site#

  1. Navigate to Hugo theme website
  2. Down your favorite theme to directory ~/Projects/<site name>/themes
  3. Modify your config file 
    baseURL = "https://<URL of your Git Hub Pages>/"
languageCode = "en-us"
title = "<website name>"
theme = "<theme name>"
  4. Run hugo server to check if everything in your local host is expected

Add Git Hub Pages repository as submodule of your source repository#

cd ~/Projects/<site name>
git submodule add -b main <URL of your Git Hub Pages> public

# Generate static resouce in public directory
hugo 
cd public
git add .
git commit -m ""Initiate a new hugo project"
git push origin main
