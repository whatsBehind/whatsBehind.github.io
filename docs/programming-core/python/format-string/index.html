    Format String in Python #

    f-strings #

    f-strings provide a concise and readable way to embed expressions inside string literals. They are prefixed with the letter f or F and use curly braces {} as placeholder for expressions

    Quick Start

    name = "Alice"
age = 30

greeting = f"Hello, my name is {name} and I am {age} years old"
print(greeting)

    Output

    Hello, my name is Alice and I am 30 years old

    Usage #

    • Expressions

      result = f"The sum of 2 and 3 is {2 + 3}."
print(result)

      Output

      The sum of 2 and 3 is 5.

    • Functions

      def greet(name):
    return f"Hello, {name}!"

message = f"{greet('Alice')}"
print(message)

      Output

      Hello, Alice!

    .format() #

    .format() is a method of str. It uses curly braces {} as placeholders within the string, which are replaced by the values passed to the format method.

    Quick View

    name = "Alice"
age = 30

greeting = "Hello, my name is {} and I am {} years old.".format(name, age)
print(greeting)

    Output

    Hello, my name is Alice and I am 30 years old.

    Usages #

    • Positional Arguments: Values are inserted into the placeholders in the order they appear. Values can be used more than once.

      string = "{0} is {1} years old. {0} wants to eat banana".format("Tom", 30)
print(string)

      Output

      Tom is 30 years old. Tom wants to eat banana

    • Named Arguments: You can use named placeholders and pass values using keyword. Values can be reused also.

      string = "{name} is {age} years old. {name} wants to eat banana".format(name="Tom", age=30)
print(string)

      Output

      Tom is 30 years old. Tom wants to eat banana

    Modifiers #

    Modifiers in Python string formatting allow you to control the way your data is presented in strings. Both f-strings and the .format() method support a variety of modifiers for formatting numbers, strings, dates, and more.

    Common Modifiers

    • Width and Alignment
    • Precision
    • Type Specifiers

    Width and Alignment #

    These modifiers control the width of the field and the alignment of the content within the field.

    Syntax

    {value:width}   # width syntax

{value:<}       # left alignment
{value:>}       # right alignment
{value:^}       # center alignment

    Example

    Alignment modifier :< need to be used before width :width

    print(f"{'Alice':<10}")     # "Alice     "
print(f"{'Alice':>10}")     # "     Alice"
print(f"{'Alice':^10}")     # "  Alice   "

    Precision #

    Precision is mainly used with floating-point numbers to control the number of digits after the decimal point.

    Syntax

    {value:precisionf}

    **Example

    print(f"{3:.2f}")           # "3.00"
print("{:.2f}".format(3))   # "3.00"

# Combined with width and alignment
print(f"{3:<10.2f}")        # "3.00      "
print(f"{3:>10.2f}")        # "      3.00"

    Type Specifier #

    Type specifiers allow you to format numbers in different bases (binary, octal, hexadecimal), percentages, and more.

    Example

    # Integer types
number = 255
print(f"{number:b}")                # Binary: "11111111
print(f"{number:o}")                # Octal: "377"
print(f"{number:x}")                # Hexadecimal: "ff"
print("{:b}".format(number))        # Binary: "11111111"
print("{:o}".format(number))        # Octal: "377"
print("{:x}".format(number))        # Hexadecimal: "ff"

# Percentage
percentage = 0.85
print(f"{percentage:.2%}")          # "85.00%"
print("{:.2%}".format(percentage))  # "85.00%"
