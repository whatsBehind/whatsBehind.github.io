    re in Python #

    re.search() #

    The re.search() method is used to search a string for a match to a regular expression pattern. It scans through the string from left to right and returns the FIRST match it finds. If a match is found, it returns a match object; otherwise, it returns None.

    Method Signature # 

    search(pattern: str, string: str, flags: int=0) -> re.Match
    • pattern: The regular expression pattern to search for.
    • string: The string to search within.
    • flags: Optional flags to modify the behavior of the pattern (default is 0).

    Match Object Methods #

    If a match is found, re.search() returns a match object. This object provides several useful methods and attributes:

    • .group() -> str: Returns the string matched by the regular expression.
    • .start() -> int: Returns the starting position of the match.
    • .end() -> int: Returns the ending position of the match.
    • .span() -> turple(int): Returns a tuple containing the (start, end) positions of the match.

    Example # 

    import re

pattern = r"\d+"
input_string = "There are 123 numbers in 456 text"

match = re.search(pattern, input_string)
print(f"`re.search()` type -> {type(match)}")

if match:
    print(f"`match.group()` type -> {type(match.group())}")
    print(f"`match.group()` value -> {match.group()}")
    print(f"`match.start()` type -> {type(match.start())}")
    print(f"`match.start()` value -> {match.start()}")
    print(f"`match.span()` type -> {type(match.span())}")
    print(f"`match.span()` value -> {match.span()}")
else:
    pass

    Output

    `re.search()` type -> <class 're.Match'>
`match.group()` type -> <class 'str'>
`match.group()` value -> 123
`match.start()` type -> <class 'int'>
`match.start()` value -> 10
`match.span()` type -> <class 'tuple'>
`match.span()` value -> (10, 13)

    re.match() #

    The re.match() method is used to determine if the regular expression pattern matches AT THE BEGINNING of the string. If a match is found at the beginning of the string, it returns a match object. Otherwise, it returns None.

    Method Signature # 

    match(pattern: str, string: str, flags: int=0) -> re.Match

    Example # 

    import re

pattern = r"word"
word_at_beginning = "word is at the beginning of the sentence"
word_not_at_beginning = "The word is not at the beginning of the sentence"

match = re.match(pattern, word_at_beginning)
non_match = re.match(pattern, word_not_at_beginning)

if match:
    print(f"Matching string -> {match.group()}")
else:
    pass

if non_match:
    pass
else:
    print(f"word is not found at the beginning")

    Output

    Matching string -> word
word is not found at the beginning

    re.fullmatch() #

    The re.fullmatch() method is used to check if the entire string matches a regular expression pattern. It differs from re.match() in that re.fullmatch() requires the whole string to match the pattern, not just a prefix. If the entire string matches the pattern, it returns a match object; otherwise, it returns None

    Method Signature # 

    fullmatch(pattern: str, string: str, flags: int=0) -> re.Match

    Example # 

    import re

pattern = "word"
full_match_string = "word"
non_full_match_string = "words"

match = re.fullmatch(pattern, full_match_string)
non_match = re.fullmatch(pattern, non_full_match_string)

if match:
    print("Found full match")
else:
    pass

if non_match:
    pass
else:
    print("No full match")

    Output

    Found full match
No full match

    re.findall() #

    Method Signature #

    The re.findall() method is used to find all occurrences of a pattern in a string. It returns a list of all matches found. If no matches are found, it returns an empty list. Unlike re.search() and re.match(), which return match objects, re.findall() directly returns the matched strings.

    findall(pattern: str, string: str, flags: int=0) -> list(str)

    Example # 

    import re

pattern = r"\bword\b"
text = "word in a wordy word sentence with words"

matches = re.findall(pattern, text)
print(f"Type of `re.findall()` -> {type(matches)}")
print(f"Type of item in matches -> {type(matches[0])}")
print("All matches found:", matches)

    Output

    Type of `re.findall()` -> <class 'list'>
Type of item in matches -> <class 'str'>
All matches found: ['word', 'word']
